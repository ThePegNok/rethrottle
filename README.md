	# ReThrottle

ReThrottle is a simple mod for Minecraft 1.20 Forge with the sole purpose of balancing the minecart speed in a vanilla-friendly way by increasing the speed cap from 8 m/s to 16 m/s.

The top speed can also be customized to any value greater than zero in the config.

Original by DuckyCrayfish on [CurseForge](https://www.curseforge.com/minecraft/mc-mods/throttle).
